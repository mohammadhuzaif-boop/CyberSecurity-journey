## Operating Systems & Linux Foundations
5 Key Jobs of an Operating System
Process Management – Starting, stopping, and multitasking applications.

Memory Management – Allocating RAM to processes and freeing it when done.

File System Management – Organizing, reading, writing, and protecting files.

Device Management – Communicating with hardware like keyboards, disks, and network cards.

User Management & Security – Controlling access, authentication, and permissions.

Important Linux Directories
Directory	Purpose
/etc	System-wide configuration files.
/var/log	Log files generated by the system and applications.
/home	Personal directories for each user.
/tmp	Temporary files; world-writable, often a security risk.
Understanding chmod 600
chmod 600 file sets permissions so only the owner can read and write.

No other users or groups can access it.

Used for sensitive files like SSH keys and credentials in cloud environments.
