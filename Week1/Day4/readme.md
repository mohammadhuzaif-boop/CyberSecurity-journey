# Day 4 – Networking Lab

## What I did
- Used TryHackMe AttackBox as my Linux environment.
- Identified network interface: `ens5`.
- Ran ping, traceroute, and nslookup to test connectivity and DNS.
- (Optional) Captured ICMP packets in Wireshark on interface `ens5`.

## Key takeaways
- Ping uses ICMP to test reachability.
- Traceroute maps every router (hop) to a destination.
- DNS translates domain names to IP addresses.
- Subnetting divides a network into smaller blocks using CIDR.
